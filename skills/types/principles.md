# Principles / Card Grid

3-4 principles, values, or key items displayed as equal cards.

## Principles

**Equal visual weight.** All cards same size, same structure. None should look more important than another (unless intentionally designed that way).

**Cards create separation.** Background fill or borders visually isolate each item. Cards feel more structured than a plain list.

**Title on top, description below.** Inside each card: read the principle name first, then the explanation below.

## Adapt

- 3 items: 3 equal columns. Cleanest layout.
- 4 items: 2×2 grid or 4 columns.
- 2 items: two-columns type is probably more appropriate.

## Avoid

- 5+ cards (they get too small)
- Different sizes or layouts per card
- Long text inside cards (keep it concise)
