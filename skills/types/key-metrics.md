# Key Metrics

Multiple KPIs at a glance. Dashboard-style overview.

## Principles

**Glanceable like a dashboard.** 2-4 metrics shown simultaneously so the audience grasps the overall picture.

**Uniform structure per metric.** Number + label + optional description. Every metric follows the same format for easy comparison.

**Numbers first, explanation second.** Numbers big and bold, labels small and light. This contrast creates scannability.

## Structure

- Left-right split: title on left, metrics stacked vertically on right
- Top title + grid: 2-4 columns of metrics below

## Avoid

- 5+ metrics on one slide (dilutes impact)
- Different layout per metric
- Metrics without numbers (text-only defeats the purpose)
