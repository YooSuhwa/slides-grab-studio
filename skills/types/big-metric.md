# Big Metric

One large number dominates the slide. KPIs, growth rates, key figures.

## Principles

**The number must be the first thing you see.** Visually the largest, heaviest element — bigger than any title. It should trigger a "wow" reaction.

**Context below the number.** "42%" alone means nothing. "Revenue growth YoY: 42%" gives it meaning.

**One number only.** For multiple numbers, use key-metrics or statistics type.

## Avoid

- Making the number body-sized (it needs to be dramatically large)
- Decorations around the number (the number itself is the visual)
- A number without context
