# Two Columns

Two panels side by side. For comparison or parallel items.

## Principles

**Best for comparison or parallel listing.** "A vs B", "pros / cons", "before / after". Simple listing also works, but the format shines when items relate to each other.

**Title on the left, grid on the right** — or title on top with a 2-column grid below.

**Uniform structure across all items.** Every item should follow the same format (bold heading + description) so scanning is effortless.

## Adapt

- 2 items: wide 2 columns, rich descriptions.
- 4 items: 2×2 grid.
- 6+ items: consider 3 columns or simple-list instead.

## Avoid

- Columns with very different lengths (visual balance matters)
- Different format per item
