# Closing Slide

The last slide. Wraps up the deck — Q&A, thank you, contact info.

## Principles

**Mirror the cover.** If the cover opened the door, the closing shuts it. Maintain similar visual weight and whitespace. The two should feel like bookends.

**Less is more.** "Thank you" or "Questions?" is enough. Add contact info small if needed.

## Adapt

- Simplest: one big line ("Thank you", "Q&A")
- With contact: title + small email/URL below
- With takeaway: one-line key message + closing line

## Avoid

- New information on the closing slide (it's a wrap-up, not more content)
- Completely different layout from cover (keep visual symmetry)
