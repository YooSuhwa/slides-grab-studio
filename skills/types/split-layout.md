# Split Layout

Text on one side, image on the other. Best when visual material is available.

## Principles

**Text explains, image shows.** They tell the same story in different ways. If text alone is enough, don't force an image.

**One side is the star.** If text matters more, give it more space. If the image is impactful, make it bigger. 50:50 emphasizes neither.

**Images can go full-bleed.** Especially photos or screenshots — pushing them to the edge with no padding increases impact.

## Variants

- Left text + right image: most natural (follows reading direction)
- Left image + right text: when you want the visual to lead
- Full-bleed image: image fills one half edge-to-edge, no padding

## Avoid

- Using split-layout without an image (use content or two-columns instead)
- Both sides being text
