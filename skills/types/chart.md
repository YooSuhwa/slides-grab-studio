# Chart

Data visualization using graphs and charts.

## Principles

**One chart, one insight.** "Revenue is growing", "A beats B". A single message per chart. Multiple insights → multiple slides.

**The title states the insight, not the description.** "Revenue grew 3 consecutive quarters" beats "Revenue trend" — the title should tell the audience what to see.

**Simpler is better.** Reduce legend items, clarify axis labels, remove unnecessary gridlines. Maximize the data-ink ratio.

## Chart type selection

- Change over time → line chart
- Item comparison → bar chart
- Proportion → donut/pie (5 items or fewer)
- Distribution → scatter plot

## Avoid

- 3D charts (distorts perception)
- 5+ legend items (hard to distinguish colors)
- Chart without a title
- Chart without an insight sentence
